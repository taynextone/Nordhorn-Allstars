# Nordhorn Allstars

Gameboy-style Basketball-RPG im Browser.

## Spielen

- Lokal: `python3 -m http.server 8080` und dann http://127.0.0.1:8080/
- GitHub Pages: https://taynextone.github.io/Nordhorn-Allstars/

## Deployment

Jeder Push auf `master` startet den Workflow `.github/workflows/deploy-pages.yml` und veröffentlicht die statischen Dateien per GitHub Pages Actions.
